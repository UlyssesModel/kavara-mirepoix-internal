# @mirepoix/cli

Mirepoix command-line entry. Wires the other three packages together and
exposes the mirepoix binary. Replaces phase-zero-spike/mirepoix-spike.ts as the
canonical execution surface once sub-phases B/C/D land.

## Planned usage

```sh
mirepoix --system-prompt-file=PATH --cwd=PATH "your prompt"
```
