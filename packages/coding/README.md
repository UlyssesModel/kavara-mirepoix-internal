# @mirepoix/coding

Mirepoix base tools per ADR-002. Four tools, no more, no permission dialogs:

- bash — run a shell command, return stdout/stderr/exit
- read — read a file, return contents
- write — write a file, create or overwrite
- edit — replace old_string with new_string in a file (must match uniquely)
